No Brainer Markdown to HTML & PDF

For each markdown files it will generate:

  • an HTML Document
  • an HTML Presentation (using reveal.js)
  • a PDF Document (using XeLaTeX)
  • a PDF Presentation (using Beamer)
./compile.sh

If you want to be the 1337, install stack and

./build.sh
./compile

Dependencies

  • pandoc -- Tested with pandoc 1.15.0.6
  • XeLaTeX -- Tested with XeTeX 3.14159265-2.6-0.99992 (TeX Live 2015)
  • metropolis Beamer theme (working forked version here: https://github.com/yogsototh/mtheme)
